Uygulama Özellikleri Ödev dökümanındaki tüm isterler başarıyla uygulanmıştır:

Oturum Yönetimi: Firebase Authentication kullanılarak güvenli giriş ve kayıt olma işlemleri.

Giriş Sayfası: Kullanıcıyı karşılayan, profil fotoğrafı ve öğrenci bilgilerini içeren özgün tasarım.

Güncel Kurlar: API üzerinden çekilen canlı döviz verileri ve yükleme ekranı (ActivityIndicator).

Haber Modülü: 5 farklı kategoride RSS haber akışı, detay sayfası ve Share API ile haber paylaşma özelliği.

Hava Durumu: API entegrasyonu, Türkçe karakter dönüşümü ve eklenen şehirlerin JSON formatında yerel depolamada saklanması.

Yapılacaklar Listesi: Firebase Realtime Database ile entegre çalışan tam CRUD (Ekle, Sil, Güncelle) sistemi.

Ayarlar: Uygulama genelinde dinamik olarak değişen Koyu Tema desteği.

Teknik Detaylar Framework: .NET MAUI (net9.0-android)

Veritabanı: Firebase Realtime Database

Kimlik Doğrulama: Firebase Auth

Veri Formatı: JSON & RSS
